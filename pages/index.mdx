# Getting Started with SaaSavant

Follow the steps below to set up your project environment.

## Prerequisites

Ensure you have the following installed on your system:

- Node.js: Version 18
- npm: Version 8
- Next.js: Version 14

## Installation

- Open your terminal

- Navigate to your project directory by running:

```bash
cd /path/to/your/project
```

Install project dependencies using npm:

```bash
npm install
```

This will install all the required dependencies for your project.

## Next Steps

1. Edit the `siteMetadata.js` file:

Open the `siteMetadata.js` file in the root of your project and fill in the missing information. Leave anything blank that is not yet applicable.

2. Edit the `.env.local` file:

Open the `.env.local` file in the root of your project and fill in the missing information. Leave anything blank that is not yet applicable.

Go to the next page to get started configuring Firebase.
